# 🔐 React Password Generator
A simple and responsive **Password Generator** built using **React, Vite, and TailwindCSS**.  
This app allows users to generate strong and secure passwords with customizable options like password length, numbers, and special characters.

---

## 🚀 Features

- Generate secure random passwords
- Adjustable password length (6 - 100)
- Option to include numbers
- Option to include special characters
- Copy password to clipboard with one click
- Clean and responsive UI
- Built using modern React hooks

---

## 🛠️ Technologies Used

- React
- Vite
- TailwindCSS
- JavaScript (ES6)
- HTML5
- CSS3
